# Simple Rule-Based Chatbot

A basic chatbot built in Python using if-else logic. No AI/ML involved — just simple pattern matching on predefined inputs.

## What it does
- Responds to greetings (hi, hello, hey)
- Answers a few fixed questions
- Exits when you type bye/exit/quit
- Runs in a loop until you exit

## How to run
```bash
python chatbot.py
```

## Concepts used
- Control flow (if-elif-else)
- Loops (while)
- String handling (.lower())

## Note
This is a rule-based bot, not a real AI. Just matches your input to fixed responses.
